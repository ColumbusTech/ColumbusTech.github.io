Columbus Engine

The biggest project

Crossplatform 3D engine written in C++. It using OpenGL (soon D3D and Vulkan) for rendering, have dynamic lighting, PBR, flexible particle simulation, rigidbody physics, advanced math library, component-system.

Columbus CMF

3D model format, add-on for Blender, C library for loading\saving and console util.

Columbus Raytracer

Simple CPU raytracer written in C++.

Columbus Serial

Simple C++ header-only library for working with computer serial (COM) port.

Columbus Filesystem

Simple C++ one-file header-only library for working with filesystem.

Columbus Info

Simple C++ one-file header-only library for getting system information like size of RAM, CPU cores count and other.

Columbus Image

Simple C library for loading/saving PNG, JPG, BMP, TGA, TIF images.

Columbus Encrypt

Simple C++ one-file header-only library which realizing some encription and decription algorithms.
