Columbus Projects

ColumbusEngine

The Greatest project

Crossplatform 3D game engine, written in C++, using OpenGL (soon Vulkan and D3D) for rendering, XML for materials, lights and scenes, CMF for models, using advanced Entity-Component System (ECS) for scene hierarchy. Have flexible Particle System

ColumbusDirectory

Simple C++ one-file header-only library for cross-platform filesystem workflow

ColumbusInfo

Simple C++ one-file header-only library for cross-platform getting system info like number of CPU cores, RAM size, free RAM size, GPU info and other

ColumbusSTD

Simple C++ header-only library like C++ STL, written for experience

ColumbusImage

Simple C library for image loading

It supports PNG, JPG, TIF, TGA, BMP image formats for read and write

ColumbusEncrypt

Simple C++ header-only library realizing some encryption algorithms

It supports XOR cipher, Ceasar cipher

ColumbusCMF
